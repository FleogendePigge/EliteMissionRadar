Elite Mission Screen Filter (v7.5)

The official name of this tool is Elite Mission Screen Filter. The current version is 7.5.

How It Works
This tool runs completely in the background. It takes localized screenshots of your game mission board dozens of times per second and uses OCR text recognition to read the text. The moment it matches your keywords, a reward threshold, or specific engineering materials, it triggers an audio alarm to alert you immediately.

Key Features
First, universal mission filter. You can customize the tool to look for anything you want, such as high-value Wing Mining Missions, tier five rare engineering materials, specific trade commodities, or simply missions with payouts above a certain credit amount.Second, dual-track input detection. It monitors keyboard inputs such as W or S, mouse side buttons, and gamepad D-Pad controls. Whenever you scroll the mission board in-game, the scanning activates instantly.Third, full hardware volume control. The main console sound system is fully synchronized with your Windows audio configuration, giving you direct control over the alert sound intensity.

Audio Telemetry Status
While you are scrolling through the mission board in-game, the background beeps will tell you the real-time status of the tool.The first sound is a low-pitched pulse. This means everything is running perfectly fast and you will not miss any missions.The second sound is a high-pitched alarm. This indicates that your PC is lagging and the scanning is running too slow. This usually happens if you turn on too many scanning filters at the same time. You should adjust your settings to prevent dropping tasks.The third sound is the target detected siren, which plays an airhorn sound. This means a target has been found. Stop scrolling immediately and accept the mission manually.

Onboarding and Usage Guide
Step one, preparation. Ensure you are on Windows 10 or 11, set the game display mode to windowed or borderless windowed, and maintain the original classic orange HUD theme. If you have modified the interface colors using EDHM or other graphics mods, the color masks will fail.
Step two, initial launch. Place the compiled executable in the same root folder as the tessdata folder and the airhorn.wav sound file. Run the program, and follow the simple 4-step setup wizard to configure your preferred speed and keyword packages.Step three, operation. Go to the in-game mission board, hold down your scrolling input to let the board drift at full speed. Listen to the background beep, and stop to accept the mission manually once the airhorn blows.

Technical Limitations
First, screen resolution bounds. The tool coordinates were mapped on a physical 1920 by 1200 monitor. While dynamic scaling is implemented, alignment cannot be guaranteed on 2K, 4K, or ultra-wide displays due to a lack of test hardware. If offsets occur, coordinates must be adjusted inside the source code function.
Second, OCR misreads. Due to pixel edge font distortion or scrolling blur, the text engine can occasionally mistake characters, such as reading the letter O as zero, or the letter I as one. Common misread variations like G0LD or S1LVER are already included, but custom entries may require adding potential variants manually.
Third, hardware load. Running fast background image processing and OCR operations requires decent processing power, and older laptops may experience lag.
